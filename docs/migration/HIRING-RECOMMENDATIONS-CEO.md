## Recommended Interview Questions

### AWS Solutions Architect

1. "Describe a complex migration you led from another cloud to AWS."
2. "How would you design a multi-region failover for our platform?"
3. "What's your approach to cost optimization on AWS?"
4. "How do you handle EKS cluster upgrades with zero downtime?"

### AI/ML Engineer

1. "Explain your experience with AWS Bedrock or similar LLM platforms."
2. "How would you optimize costs for high-volume LLM calls?"
3. "Describe an agentic AI system you've built."
4. "How do you handle prompt injection and AI safety?"

### QA Engineer

1. "How would you test 60+ API endpoints efficiently?"
2. "Describe your approach to performance testing for 100K concurrent users."
3. "How do you integrate QA into a CI/CD pipeline?"
4. "What's your experience with testing real-time WebSocket connections?"
